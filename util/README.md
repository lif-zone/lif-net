LIF - Util
==========

## Installation

```
git clone https://github.com/lif-zone/server.git lif_server
cd lif_server
npm install
```

## Node Development
```# cd lif_server
cd util

# run all tests in node (mocha)
mocha

# run specific tests
mocha -g etask``

## debugging
- ```mocha --inspect-brk```
- Open Chrome browser
- Browse to chrome://inspect
- Click 'Open dedicated DevTools for Node'
