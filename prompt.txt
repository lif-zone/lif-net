Debugged browser test failures in sequence:

3. SyntaxError from unpkg.com/chai: chai v6 is ESM-only, breaks as regular <script>. Chrome had old UMD version cached. Fixed by pinning to https://unpkg.com/chai@4/chai.js in test_util.html.
4. Added/removed CDP session, response logging, request logging during debugging, then cleaned them up.
Result: passes: 4, failures: 0, exit 0. Only 4 tests run in browser because most util/test.js tests use is_node guards.
